# Pythonlibrarycoursework

## Описание:

PythonLibraryCourseWork - это приложение для анализа транзакций, которые находятся в Excel-файле. Приложение будет генерировать JSON-данные для веб-страниц, формировать Excel-отчеты, а также предоставлять другие сервисы.

## Установка:

1. Клонируйте репозиторий:
```
https://github.com/RomanBatyuk/Python_library_course_work
```
2. Установите зависимости:
```
pip install -r requirements.txt
```

## Использование:

1. Откройте проект в PyCharm.
2. Теперь вы можете поучавствовать в разработке ;)
3. Для тестирования функционала обратитесь к директории tests в корне проекта.

## Функционал:

В директории src:

* В модуле views.py реализована главная функция, принимающая на вход строку с датой и временем в формате 
YYYY-MM-DD HH:MM:SS и возвращающую JSON-ответ и результаты модулей services.py и reports.py.

* В модуле services.py реализована функция, которая возвращает JSON со всеми транзакциями, которые относятся к переводам физлицам.

* В модуле reports.py реализована функция - декоратор для функций-отчетов, который записывает в файл результат,
  который возвращает функция, формирующая отчет.
  Функция формирующая отчет - Функция возвращает средние траты в каждый из дней недели за последние три месяца (от переданной даты).

* В модуле utils.py реализован функционал гравной функции.

* В модуле logging_config.py реализован логер для проекта, записывающий логи программы в директори logs в корне проекта.
